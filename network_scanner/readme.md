  新功能特性：

  1. 扫描所有能ping通的主机
    - 首先对整个网段进行快速ping扫描
    - 找出所有能ping通的在线主机
    - 不仅限于有SSH端口的主机
  2. 对所有在线主机进行端口扫描
    - 扫描18个常用端口（FTP、HTTP、HTTPS、MySQL、Redis等）
    - 无论是否有SSH端口都会显示
  3. 右侧边栏显示所有在线主机
    - 每个ping通的主机都会在右侧显示卡片
    - 显示IP地址、在线状态
    - 列出所有开放的端口和服务
    - 如果有SSH端口，显示SSH连接状态
    - 如果SSH连接成功，显示主机名和操作系统

  运行方式：

  python /home/ivan/app.py

  访问 http://localhost:5000

  扫描流程：

  1. Ping扫描阶段：快速并发ping整个网段，找出所有在线主机
  2. 端口扫描阶段：对每个在线主机扫描常用端口
  3. SSH测试阶段：如果发现SSH端口，尝试连接并获取系统信息

  显示信息：

  - 左侧主区域：显示扫描进度和结果列表
  - 右侧边栏：详细显示每个在线主机的信息
    - 绿色状态灯表示在线
    - 端口徽章显示所有开放端口
    - SSH连接状态和系统信息（如果适用）

  这样您可以看到网络中所有能ping通的设备及其开放的端口，不仅限于SSH设备。
